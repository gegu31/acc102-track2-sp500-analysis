# ACC102 Track 2: S&P 500 Index Trend and Volatility Analysis (2020-2024)

## 1. Problem Definition & Target Audience
This project analyzes the price trend and volatility of the S&P 500 index from 2020 to 2024. The target audience includes beginner investors, finance students, and anyone interested in market behavior during the COVID-19 pandemic.

## 2. Data Source
- **Type**: Simulated data (mimics S&P 500 index performance)
- **Time Period**: January 1, 2020 – January 1, 2024
- **Key Features**: Includes long-term growth, 2020 pandemic crash, and daily volatility

## 3. Methodology
1. Data simulation (based on real S&P 500 characteristics)
2. Data cleaning and preparation
3. Daily return calculation
4. Descriptive statistics
5. Visualization with matplotlib

## 4. Key Findings
1. The index showed a strong upward trend despite short-term volatility.
2. Extreme volatility occurred in early 2020 due to the COVID-19 pandemic.
3. The average daily return was positive, indicating long-term market growth.
4. High volatility periods correspond to major economic events.

## 5. Repository Structure
- `analysis.ipynb`: Main analysis notebook
- `sp500_simulated_data.csv`: Simulated dataset
- `sp500_price_trend.png`: Price trend chart
- `sp500_daily_returns.png`: Daily returns chart
- `README.md`: This file
